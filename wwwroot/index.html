Electric Games Api

Apiet for admin

Endepunkter

Game

  • https://localhost:7265/Game - få takk i alle spill
  • https://localhost:7265/Game/{id} - søk spill etter id
  • https://localhost:7265/Game/GetGameByTitle/{title} - søk spill etter navn
Objekt:

{
"id": 0,
"title": "string",
"platform": "string",
"releaseYear": 0,
"image": "string"
}

Character

  • https://localhost:7265/Character - få takk i alle karaktere
  • https://localhost:7265/Character/{id} - søk karakter etter id
  • https://localhost:7265/Character/GetCharacterByName/{name} - søk karakter etter navn
Objekt:

{
"id": 0,
"name": "string",
"game": "string",
"image": "string"
}

ImageUpload

  • https://localhost:7265/ImageUpload - last opp et bilde
